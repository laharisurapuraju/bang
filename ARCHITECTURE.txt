SEVERITY SCORING PIPELINE
==========================

Input: "Massive flooding and stagnant water for 5 days in slum area"

├─ STAGE 1: Tokenization & Normalization
│  └─ "massive flooding and stagnant water for 5 days in slum area"
│
├─ STAGE 2: Keyword Detection
│  ├─ Keywords Found: [massive(extreme), flooding(high), stagnant(high)]
│  └─ Base Score: 6.0 (from highest severity)
│
├─ STAGE 3: Duration Analysis
│  ├─ Pattern: "for 5 days"
│  ├─ Days: 5 (>= 5)
│  └─ Duration Factor: 2.0x
│
├─ STAGE 4: Location Detection
│  ├─ Pattern Found: "slum"
│  └─ Location Factor: 1.3x
│
├─ STAGE 5: Health Impact Analysis
│  ├─ Health Keywords: None found
│  └─ Health Factor: 1.0x
│
├─ STAGE 6: Text Length Assessment
│  ├─ Word Count: 11
│  └─ Length Factor: 1.2x
│
└─ STAGE 7: Final Calculation
   ├─ Base Score: 6.0
   ├─ Total Multiplier: 2.0 × 1.3 × 1.0 × 1.2 = 3.12
   ├─ Raw Score: 6.0 × 3.12 = 18.7
   └─ Final Score: 10.0 (capped at 10.0)
